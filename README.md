# 🎬 Movie Ticket Management System

A simple **console-based Movie Ticket Management System** developed using **Core Java**. This project demonstrates object-oriented programming concepts and provides a basic ticket booking workflow through a menu-driven interface.

## 📌 Features

* 🔐 User Login
* 🎥 View Available Movies
* 🎫 Book Movie Tickets
* 📄 View Booking Details
* ❌ Cancel Booking
* 🚪 Exit Application

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Console-Based User Interface

## 📂 Project Structure

```
Movie-Ticket-Management-System/
│── Main.java
│── Movie.java
│── Booking.java
└── User.java
```

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Movie-Ticket-Management-System.git
```

### 2. Navigate to the Project Folder

```bash
cd Movie-Ticket-Management-System
```

### 3. Compile the Project

```bash
javac *.java
```

### 4. Run the Application

```bash
java Main
```
